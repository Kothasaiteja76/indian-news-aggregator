# Indian News Aggregator 📰

A modern, responsive news aggregator that fetches real-time headlines from major Indian news sources. Built with React, TypeScript, and Tailwind CSS.

## Features

- 📰 *Real-time News* - Live headlines from multiple Indian newspapers
- 🔍 *Smart Search* - Search across all news sources
- 📝 *Personal Notes* - Save notes on articles
- 🌙 *Dark Mode* - Toggle between light and dark themes
- 📄 *PDF Export* - Download news summaries as PDF
- 🎯 *Categories* - Business, Technology, Sports, Entertainment, Health, Science

## Tech Stack

- *Frontend:* React, TypeScript, Tailwind CSS
- *Build Tool:* Vite
- *APIs:* GNews, NewsData.io, Mediastack
- *Storage:* LocalStorage for notes

## Getting Started

1. Clone the repository
2. Install dependencies: npm install
3. Add API keys in .env file
4. Run: npm run dev
5. Open: http://localhost:5173
